# perpustakaan-laravel-api
# Sistem Informasi Manajemen Perpustakaan Berbasis REST API

Project ini merupakan aplikasi REST API sederhana menggunakan framework Laravel
yang dibuat untuk memenuhi tugas mata kuliah pengembangan aplikasi web.
Aplikasi ini digunakan untuk mengelola data buku perpustakaan.

## Tujuan
Membangun RESTful API yang dapat digunakan untuk melakukan pengelolaan data
buku dengan operasi Create, Read, Update, dan Delete (CRUD).

## Fitur Aplikasi
- Menampilkan seluruh data buku
- Menampilkan detail data buku
- Menambahkan data buku
- Mengubah data buku
- Menghapus data buku

## Endpoint REST API
| Method | Endpoint           | Keterangan                  |
|--------|--------------------|-----------------------------|
| GET    | /api/books         | Menampilkan semua data buku |
| GET    | /api/books/{id}    | Menampilkan detail buku     |
| POST   | /api/books         | Menambahkan data buku       |
| PUT    | /api/books/{id}    | Mengubah data buku          |
| DELETE | /api/books/{id}    | Menghapus data buku         |

## Teknologi yang Digunakan
- PHP
- Laravel Framework
- REST API
- MySQL
